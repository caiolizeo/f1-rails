
# F1 Rails

## 🖥️ Sobre o projeto:

Projeto em Rails 7 criado para visualizar e fazer apostas referentes ao campeonato atual de Formula 1, consumindo APIs do site http://ergast.com/
<br>
<br>

## ⚙️Configurando o projeto

```
ruby bin/setup
```

## 🚀 Rodando o projeto
``` 
rails s
```

## 🧪 Executando testes
``` 
rspec
```
